# devtools-reports

Jeg dokumenterer konkrete DevTools‑advarsler og browser‑breakage fra produktionssider.  
Formålet er at levere reproducerbare console‑uddrag og korte reproduktionsnoter, ikke at debugge.

## Aktuelt
- WICG issue: overflow: visible på <img> påvirker Microsoft Careers (WICG issue #253)

## Hvordan jeg arbejder
1. Reproducer i DevTools.  
2. Kopiér console og relevante DOM‑udsnit.  
3. Opret issue med artefakter.

Kontakt: åben et issue i dette repo.
